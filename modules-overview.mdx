## Managing modules

All modules are managed from a single location: **Home â†’ Setup â†’ Modules/Applications**.

Each module has a toggle switch. Turning a module on:

- Adds its menu entries to the navigation bar
- Creates any required database tables
- Makes its permissions available to assign to user groups

Turning a module off hides everything related to it without deleting any data. You can re-enable the module later and all previous records remain intact.
