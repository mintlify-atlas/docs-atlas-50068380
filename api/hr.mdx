---
title: "REST API endpoints for HR and leave management"
description: "GET, POST, PUT, DELETE /holidays and /expensereports — manage employee leave requests and expense reports via the Dolibarr REST API."
---

The HR APIs cover leave management and expense reports, letting HRIS systems sync data with Dolibarr. Use the holidays API to submit and approve leave requests, and the expense reports API to capture employee expenses through the approval workflow.

**Base paths:**
- `/api/index.php/holidays`
- `/api/index.php/expensereports`

---

## Leave requests (holidays)

### List leave requests

`GET /holidays`

<ParamField query="fk_user" type="integer">
  Filter by employee user ID.
</ParamField>

<ParamField query="status" type="integer">
  Filter by leave request status: `1` draft, `2` validated (submitted), `3` approved, `4` refused.
</ParamField>

<ParamField query="sqlfilters" type="string">
  Additional SQL filter criteria. Example: `(t.date_debut:>:'2024-01-01')`
</ParamField>

<ParamField query="sortfield" type="string" default="t.rowid">
  Field to sort results by.
</ParamField>

<ParamField query="sortorder" type="string" default="ASC">
  Sort direction: `ASC` or `DESC`.
</ParamField>

<ParamField query="limit" type="integer" default="100">
  Maximum number of results to return.
</ParamField>

<ParamField query="page" type="integer" default="0">
  Page number (zero-indexed).
</ParamField>

```bash
curl -X GET "https://your-dolibarr.example.com/api/index.php/holidays?fk_user=7&status=2" \
  -H "DOLAPIKEY: your-api-key"
```

---

### Create a leave request

`POST /holidays`

<ParamField body="fk_user" type="integer" required>
  ID of the employee requesting leave.
</ParamField>

<ParamField body="date_debut" type="integer" required>
  Start date of the leave as a Unix timestamp.
</ParamField>

<ParamField body="date_fin" type="integer" required>
  End date of the leave as a Unix timestamp.
</ParamField>

<ParamField body="fk_type" type="integer">
  Leave type ID (references the leave type dictionary, e.g. annual leave, sick leave).
</ParamField>

<ParamField body="description" type="string">
  Reason or description for the leave request.
</ParamField>

```bash
curl -X POST "https://your-dolibarr.example.com/api/index.php/holidays" \
  -H "DOLAPIKEY: your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "fk_user": 7,
    "date_debut": 1719792000,
    "date_fin": 1720051200,
    "fk_type": 1,
    "description": "Annual leave"
  }'
```

<ResponseField name="id" type="integer">
  ID of the newly created leave request.
</ResponseField>

</ResponseField>

</ResponseField>

</ResponseField>

<ParamField path="id" type="integer" required>

### Create an expense report

`POST /expensereports`

<ParamField body="fk_user_author" type="integer" required>
  ID of the employee submitting the expense report.
</ParamField>

<ParamField body="date_debut" type="integer" required>
  Start date of the expense period as a Unix timestamp.
</ParamField>

<ParamField body="date_fin" type="integer" required>
  End date of the expense period as a Unix timestamp.
</ParamField>

<ParamField body="lines" type="array">
  Array of expense lines. Each line must include `date`, `fk_c_type_fees`, `qty`, `value_unit`, and `vatrate`.
</ParamField>

```bash
curl -X POST "https://your-dolibarr.example.com/api/index.php/expensereports" \
  -H "DOLAPIKEY: your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "fk_user_author": 7,
    "date_debut": 1704067200,
    "date_fin": 1704153600,
    "lines": [
      {
        "date": 1704067200,
        "fk_c_type_fees": 2,
        "qty": 1,
        "value_unit": 85.00,
        "vatrate": 20,
        "comments": "Hotel accommodation - client visit"
      }
    ]
  }'
```

<ResponseField name="id" type="integer">
  ID of the newly created expense report.
</ResponseField>

---

<ParamField path="id" type="integer" required>

</ParamField>

<ResponseField name="id" type="integer">

</ResponseField>

<ResponseField name="fk_user_author" type="integer">

<ResponseField name="date_fin" type="integer">

<ResponseField name="total_ht" type="number">

<ResponseField name="total_ttc" type="number">

</ResponseField>

<ResponseField name="statut" type="integer">
  Report status: `0` draft, `2` validated (submitted), `5` approved, `6` paid.
</ResponseField>

<ResponseField name="lines" type="array">
  Array of expense line objects.

  <Expandable title="Line fields">
    <ResponseField name="id" type="integer">

    <ResponseField name="fk_c_type_fees" type="integer">

    </ResponseField>

    </ResponseField>

    <ResponseField name="tva_tx" type="number">

### Add an expense line

`POST /expensereports/{id}/lines`

</ParamField>

<ParamField body="date" type="integer" required>
